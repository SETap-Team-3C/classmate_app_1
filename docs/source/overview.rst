Overview
========

Classmate App 1 is a Flutter-based cross-platform application designed as a 
starter template for building educational or communication-based apps.

Key Technologies
----------------
- Flutter (Dart)
- Platform support: Android, iOS, Web, Windows, macOS, Linux
- CMake, Swift, C++ (platform-specific code)

Purpose
-------
This project serves as a foundation for building a classroom communication 
or collaboration app. It includes the default Flutter structure and can be 
extended with chat, authentication, and real-time features.
