# Developer Guide

This guide helps developers contribute to the Classmate App.

## Adding a New Screen
1. Create a new Dart file in `lib/`
2. Add a widget class
3. Register route in `main.dart`

## State Management
Planned options:
- Provider
- Riverpod
- Bloc

## Adding Features
- Create a folder under `lib/features/`
- Add UI, logic, and models
