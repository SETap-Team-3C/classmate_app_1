# Classmate App Project Summary

## Title
Classmate App — Student Messaging and Communication Platform

## Short description
Classmate is a Flutter and Firebase app that lets students sign in, chat in real time, manage profiles, call contacts, and customize the app theme.

## Main objectives

- Build a practical student communication app
- Use Firebase Authentication and Firestore effectively
- Apply secure Firestore rules
- Improve usability with clean dialogs, settings, and message actions
- Keep the codebase testable and analyzer-clean

## Features implemented

- User authentication
- Real-time direct messaging
- Message edit, delete, star, and read status
- Typing indicator
- Image attachments through Firebase Storage
- Profile editing and avatar upload
- Contact calling
- Theme persistence

## Technology stack

- Flutter
- Dart
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
- Shared Preferences
- Image Picker

## Usability improvements

- Clear message status feedback
- Safer delete and edit actions
- Cleaner settings and dialog flows
- Better error handling for Firebase issues
- More consistent UI behavior

## Security work

- Firestore rules for users, chats, and messages
- Access restricted to signed-in users
- Message updates limited to valid chat participants

## Testing and validation

- Analyzer issues cleaned up
- Widget and unit test files included
- Messaging flow validated in the app during development

## Suggested marks-friendly presentation points

1. Problem solved: student communication in one app
2. Technical depth: Firebase + Firestore + Storage
3. Security: custom Firestore rules
4. Usability: polished UI and message actions
5. Maintainability: separated screens, models, and services
