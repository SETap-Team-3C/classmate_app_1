Folder Structure
================

The project contains the following directories:

lib/
    Contains all Dart source code.

android/
    Native Android configuration and build files.

ios/
    Swift/Objective-C platform code.

web/
    Web-specific assets and bootstrap files.

windows/, macos/, linux/
    Desktop platform code.

test/
    Unit and widget tests.

pubspec.yaml
    Project metadata and dependencies.
