Installation
============

Prerequisites
-------------
- Flutter SDK (3.x or later)
- Android Studio or VS Code
- Git

Steps
-----
1. Clone the repository::

      git clone https://github.com/SETap-Team-3C/classmate_app_1

2. Navigate into the project::

      cd classmate_app_1

3. Fetch dependencies::

      flutter pub get

4. Run the app::

      flutter run
